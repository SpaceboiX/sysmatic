Information

Below is a list of commands & examples

/sc [id] [name]


ID = ID for reference
Name = Name of alter

System Create
Used to create an alter

ID is usually a shorter name of the alter

/se [id] [option] [value/image]

ID = ID of alter
Option = id, name, description, pronouns, image, color
Value = Change value | Image = Image Attachment

System Edit
Edits the specified alter

/sm [id] [message]

ID = ID of alter
Message = Message to send

System Message
Sends a message as an alter

/sp [id]

ID = ID of alter

System Profile
Shows the profile of a specific alter

/sl

System List
Lists your added alters

/si

System Information
Displays information of your system
