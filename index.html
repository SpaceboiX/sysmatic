Install Sysmatic

Information

Below is a list of commands

/sc [id] [name]


ID = ID for reference
Name = Name of alter

System Create
Used to create an alter

ID is usually a shorter name of the alter

/se [id] [option] [value]

ID = ID of alter
Option = id, name, description, pronouns, image, color
Value = Change value

System Edit
Edits the specified alter

/sr [id]

ID = ID of alter

System Remove
Remove an alter

/sf [id]

ID = ID of alter

System Front
Set the fronting alter

/sm [id] [message]

ID = ID of alter
Message = Message to send

System Message
Sends a message as an alter

/sqm [message]

Message = Message to send

System Quick Message
Send a message as the fronting alter

/sp [id]

ID = ID of alter

System Profile
Shows the profile of a specific alter

/sl

System List
Lists your added alters

/si

System Information
Displays information about your system
